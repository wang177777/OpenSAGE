# Public release notes - 20 July 2026

## Release scope

This repository brings together the research software, public aggregate/output-level data, submitted figures, and reproducibility resources accompanying the OpenSAGE/DGQC study.

The release contains:

- output-level analysis records for 56 OpenSAGE drafts;
- output-level analysis records for 70 direct-generation drafts across seven systems and ten matched diseases;
- Supplementary Data 1-3 and CSV exports of Tables T001-T002 and S01-S14;
- the DGQC scoring instrument and operational manual;
- analysis, figure-generation, manifest, and validation scripts.

## Analysis provenance

The public tables retain output identifiers, analysis-set membership, system labels for the matched comparison, reported scores, and critical-defect status. Historical generation inputs and evaluated document files are maintained in the controlled research archive.

## Evaluation and analysis definition

- The 1,512 ratings were provided by twelve expert reviewers in one blinded expert-rating round.
- The final DGQC score is the primary continuous endpoint for all 126 outputs.
- Reviewer-mean DGQC and raw D1-D7 summaries are derived from the same rating round; Stage 1 and Stage 2 denote analysis subsets.

## Public release boundary

The public files support the released descriptive summaries, matched comparisons, exact sign-flip tests, DGQC weighting, and figure inputs. Rating-level expert records, reviewer identifiers, identity mappings, free-text comments, and confidential mapping files remain in the controlled research dataset.
